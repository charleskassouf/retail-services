# Retail App

Is an application to Calculate discount by type.

### Prerequisites

- Java 8 and above
- Maven

### Installing

######Checkout the project from git

```
git clone https://github.com/charleskassouf/retail-services.git
```

######To run the application
```
mvn spring-boot:run
```

######To run tests

```
mvn clean install
```
